find("selector") : 선택한 엘리먼트의 자손들 (겹겹이 껴있는 것까지 포함.)중에, 탐색하고자 하는 엘리먼트를 검색한다.
children("selector") : 선택한 엘리먼트의 자식요소들 중에 (바로 아래 요소)탐색하고자 하는 엘리먼트를 검색한다.
parent()해당요소의 바로 위에 존재하는 하나의 요소를 반환합니다. / parents("selector") 만약 하나가 아닌 모든 부모 요소를 반환할 필요가 있다면.: 선택한 엘리먼트의 부모요소를 검색
next("selector") : 선택한 엘리먼트 다음에 나오는 요소를 검색
